# VIGILA - Seu Guia de Bolso de Segurança Pessoal e Proteção

O **VIGILA** é uma plataforma moderna de segurança pessoal que oferece ferramentas de monitoramento, perfis de segurança personalizados e gestão de acessos avançada. Desenvolvido para oferecer tranquilidade e proteção na palma da sua mão.

## 🚀 Funcionalidades Principais

- **Monitoramento em Tempo Real:** Acompanhamento de status e alertas de segurança.
- **Perfis de Segurança:** Escolha entre diferentes perfis (Padrão e PRO) para adaptar a proteção às suas necessidades.
- **Gestão de Acessos PRO:** Sistema administrativo exclusivo para conceder acessos temporários (7 dias, 1 ano) ou vitalícios.
- **Interface Moderna:** UI intuitiva construída com Tailwind CSS e animações fluidas com Framer Motion.
- **Segurança Robusta:** Autenticação e banco de dados em tempo real via Firebase (Auth & Firestore).

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React 18, TypeScript, Vite.
- **Estilização:** Tailwind CSS.
- **Animações:** Framer Motion.
- **Ícones:** Lucide React.
- **Backend:** Firebase (Authentication, Firestore).

## ⚙️ Configuração do Ambiente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/kravmagaipiranga/vigila.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configuração do Firebase:**
   Certifique-se de que o arquivo `firebase-applet-config.json` esteja presente na raiz do projeto com as credenciais corretas do seu projeto Firebase.

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

## 🛡️ Administração

O painel administrativo é acessível exclusivamente para o administrador principal (`kravmagaipiranga@gmail.com`) através da aba de **Ajustes**, permitindo a gestão completa de licenças PRO:
- Concessão de acesso por **7 dias**.
- Concessão de acesso por **1 ano**.
- Concessão de acesso **vitalício**.
